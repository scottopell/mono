Observations & Analysis

This visualizer tracks how bubble sort affects elements with different color properties. Key metrics:

  • Color Position: Average index position of red vs green elements over time
  • Clustering Index: How grouped together same-colored elements are (0 = dispersed, 1 = fully clustered)
  • Color Inversions: Pairs where larger value comes before smaller, split by cross-color vs same-color
  • Swap Ratio: Proportion of swaps between different colors vs same colors
